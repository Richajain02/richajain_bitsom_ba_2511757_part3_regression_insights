# Retail Store Sales Regression Analysis

## Objective

The objective of this project is to identify the factors most strongly associated with monthly sales performance across retail stores using regression analysis.

---

## Dependent Variable

The dependent variable (target variable) for the regression analysis is:

* monthly_sales

This represents the monthly sales revenue generated by each store and is the business outcome we want to explain and predict.

---

## Potential Independent Variables

The following variables may influence monthly sales and can be used as predictors in regression models:

* marketing_spend
* footfall
* avg_discount_pct
* staff_count
* inventory_availability_pct
* competitor_distance_km
* holiday_flag
* customer_rating
* region (categorical)
* store_type (categorical)

---

## Numerical Variables

The following variables are numerical and can be used directly in regression analysis:

* marketing_spend
* footfall
* avg_discount_pct
* staff_count
* inventory_availability_pct
* competitor_distance_km
* customer_rating
* monthly_sales
* monthly_profit

---

## Categorical Variables

The following variables are categorical and require conversion into dummy variables before inclusion in regression models:

* region

  * East
  * West
  * North
  * South

* store_type

  * Hypermarket
  * Supermarket
  * Convenience
  * Express

---

## Variables That May Require Cleaning or Transformation

### Missing Values

The dataset contains missing values in:

* competitor_distance_km (6 missing records)
* customer_rating (8 missing records)

These values should be handled before regression analysis, either through imputation or removal of affected records.

### Date Variable

* month

This variable is stored as a date field. It may be used for trend analysis but is not required for the primary regression model.

### Categorical Encoding

The following variables must be converted into dummy variables:

* region
* store_type

To avoid multicollinearity, one category from each variable will be used as the reference category.

---

## Variables That May Not Be Useful for Regression

### store_id

* Unique identifier for each store.
* Does not provide explanatory business value.
* Should not be included as a predictor in regression models.

### monthly_profit

* Strongly related to sales performance.
* Using it as a predictor may create interpretation issues because profit is partly derived from sales.
* It will not be included in the primary sales prediction model.

---

## Initial Regression Candidates

The strongest business candidates for regression analysis are:

* marketing_spend
* footfall
* inventory_availability_pct
* avg_discount_pct
* customer_rating
* holiday_flag
* staff_count

These variables are directly controllable by management and may provide actionable business insights.
